import { LoginButton } from './login-button';

This pre-styled button can be used by third parties to kick off the passport 0xAuth Flow.

### Component usage

```js
<LoginButton href={'https://passport.kubelt.com?appid=<appid>'} />
```

### Render hello world!

```js live
<LoginButton href={'https://passport.kubelt.com?appid=<appid>'} />
```
